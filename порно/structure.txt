Мини-отель - Структура страниц

1. Страница входа (index.html)
   - Форма авторизации
     * Поле для логина
     * Поле для пароля
     * Кнопка "Войти"
   - Копирайт

2. Панель менеджера (manager.html)
   - Шапка с именем менеджера
   - Три вкладки:
     а) Сотрудники
        * Таблица сотрудников (ФИО, Должность, Логин, Статус, Действия)
        * Кнопка добавления сотрудника
     б) Заказы
        * Таблица заказов (Номер, Дата/время, Номер комнаты, Услуга, Статус, Сотрудник)
     в) Смены
        * Таблица смен (Дата, Время, Список сотрудников, Действия)
        * Кнопка создания смены

3. Обслуживание номеров (room_service.html)
   - Шапка с именем сотрудника и временем смены
   - Статистика (Активные заказы, Выполнено сегодня)
   - Фильтры (Статус, Дата)
   - Таблица заказов
     * Номер комнаты
     * Время
     * Тип уборки
     * Пожелания
     * Приоритет
     * Статус
     * Действия
   - Форма нового заказа

4. Услуги отеля (hotel_services.html)
   - Шапка с именем сотрудника
   - Статистика (Активные, Выполнено)
   - Таблица услуг
     * Номер
     * Описание
     * Статус
     * Действия

Общие элементы:
- Хедер (header.html)
  * Логотип отеля
  * Навигация
  * Информация о пользователе
  * Кнопка выхода
